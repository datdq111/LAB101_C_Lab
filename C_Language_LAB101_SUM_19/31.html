Convert decimal to binary. C.S.P0031
Approved Short Assignment
C Basic Manual Grading Design a program that allows users to input a positive integer and output the corresponding binary form.

In mathematics and digital electronics, a binary number is a number expressed in the binary numeral system or base-2 numeral system which represents numeric values using two different symbols: typically 0 (zero) and 1 (one). The base-2 system is a positional notation with a radix of 2. Because of its straightforward implementation in digital electronic circuitry using logic gates, the binary system is used internally by almost all modern computers and computer-based devices. Each digit is referred to as a bit.

  • The program should be repetitive until users close the program

Function details:

Function 1: Display a screen to prompt users to input a positive decimal number.

  • Users run the program, display a screen to ask users to enter a positive decimal number.
  • Users input a positive decimal number. Then, perform Function 2.

Function 2: Convert decimal to binary.

  • The program converts the inputted decimal to coresponding binary, and returns an array containing those converted bits from left to right.
  • Perform Function 3.

Function 3: Output bits stored in the array to the screen.
